
# Home Module
## Purpose
   - The Home page provides a global overview of the integrated dataset. Users can quickly understand:
   + Disease coverage;
   + Sample distribution;
   + Cell counts;
   + Data scale.

# Query Module
## Purpose
   - The Query module is the core functionality of Pheat. Users can explore gene expression patterns across:
   + Cell types;
   + Tissue origins;
   + Disease states;
   + T-cell subpopulations.

# Method Module
## Purpose
  - The Method page documents the computational framework used to generate the atlas.This section ensures transparency, reproducibility, and scientific rigor.

# Contact Module
## Purpose
  - The Contact page provides communication channels for users. Users may contact the development team regarding: Technical issues, Bug reports, Feature requests, Collaboration opportunities.

# About Module

  - The About page provides background information about Pheat.
